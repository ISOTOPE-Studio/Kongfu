# 江湖绝学
- 带GUI界面
- 通过指令打开GUI界面里面点击学习技能
- 要求每个门派打开的指令不同，权限要求也不同
- 每个玩家最多学习10种技能，要求有重置技能选项
- 技能要可以升级(每种技能可提升20级)，每种技能升级需求金钱及经验及等级要求
- 制作特殊物品(技能书)把技能绑在上面，要设置学习条件：等级及经验，通过此物品来打开GUI选项，同上要有升级技能的选项

## 江湖绝学GUI
